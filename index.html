1. Contraste y legibilidad

- Usa colores con buen contraste entre texto y fondo. - Emplea tipografías claras y de tamaño adecuado.

2. Navegación intuitiva

- Estructura clara con menús y enlaces visibles. - Usa etiquetas descriptivas para botones y enlaces.

3. Contenido accesible

- Proporciona texto alternativo para imágenes. - Usa subtítulos y transcripciones para contenido multimedia.

4. Diseño adaptable

- Asegúrate de que el sitio se vea bien en dispositivos móviles. - Utiliza un diseño responsivo que se ajuste a diferentes tamaños de pantalla.

5. Evitar contenido parpadeante

- Evita animaciones que puedan causar molestias o distracciones. - Usa transiciones suaves y evita el parpadeo rápido.

6. Formularios accesibles

- Etiqueta claramente los campos de formulario. - Proporciona mensajes de error claros y accesibles.

7. Uso de ARIA
